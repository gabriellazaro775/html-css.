Listas ordenadas

para criar uma lista ordenada usa-se a tag "ol" e dentro da tag usa outra tag "li" para cada item da lista

  1. acordar
  2. tomar banho
  3. vestir o uniforme
  4. ir pro hotel
  5. ir pra casa
  6. estudar html e css

    7. foi usado a tag "ol" para indicar que é uma lista ordanada e dentro da tag ol usa a tag "li" para ordenar a lista por numeração.
